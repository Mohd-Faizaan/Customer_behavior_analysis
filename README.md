# Customer_behavior_analysis

## Overview
This project focuses on analyzing customer behavior using **Python, SQL, and Power BI**.  
The goal is to extract actionable insights from raw data through data cleaning, exploratory data analysis (EDA), database querying, and dashboard visualization.

Final outputs include:
- Power BI dashboard
- Written report
- Presentation

---

## Dataset
The dataset contains customer-level data such as:
- Demographics (e.g., age, gender)
- Purchase information
- Review ratings
- Subscription and discount details

Data is loaded and processed using Python before being stored and queried from a SQL database.

---

## Tools & Technologies
- **Python** (Pandas, NumPy, Matplotlib, Seaborn)
- **SQL** (PostgreSQL / MySQL / SQL Server)
- **Power BI** (Dashboards & visualizations)
- **Jupyter Notebook** (EDA & data cleaning)
- **Gamma** (Presentation creation)

---

## Project Steps
1. **Data Loading**
   - Load raw dataset into Python using Pandas.

2. **Exploratory Data Analysis (EDA)**
   - Analyze distributions and trends.
   - Identify missing values and outliers.

3. **Data Cleaning & Preprocessing**
   - Handle missing values.
   - Create derived features.
   - Prepare data for analysis.

4. **SQL Analysis**
   - Store cleaned data in a SQL database.
   - Run analytical SQL queries.

5. **Dashboard Development**
   - Build an interactive Power BI dashboard.

6. **Reporting & Presentation**
   - Summarize insights in a report.
   - Create a presentation using Gamma.
